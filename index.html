Basic knowledge for TeX (TeX works)

  • 「$ sudo apt get texworks」で使えるようになる。

  • 「.tex」の拡張子のファイルは、「platex」というコマンドでコンパイルでき、コンパイル後のファイルの拡張子が「.dvi」>になる。

  • 「platex」というコマンドを使うには、「$ sudo apt get texlive-lang japanese」を実行する必要があった。

  • 「.dvi」の拡張子のファイルは、「dvipdfmx」というコマンドでpdf化することができる。

  • ターミナルからコマンドを打ち込んでpdfファイルを表示するには、「evince」コマンドを使う。

  • 括弧の種類「https://math-note.xyz/latex/symbols/brackets/」

  • 定義の記号:=は\usepackage{mathtools}で定義されている\coloneqqを使う。

  • ポインタ変数は、アドレスを記録しておく住所録のような機能を持つ変数で、「*」を前につける。

  • 変数の前に「&」を付けることで、変数の値ではなくそのアドレスを意味するようになる。

  • dataと&data[0]は同じ意味！

  • data+1と&data[1]は同じ意味！
